# Python Foundations Assignment

## Overview
This repository contains simple Python programs created as part of a Python Foundations assignment.  
Each program demonstrates basic programming concepts such as strings, loops, and functions.

---

## Contents

### Level 1: Reverse a String
- Reverses a user input string using slicing.

### Level 2: Count Vowels in a String
- Counts the number of vowels in a given string.

### Level 3: CLI Calculator
- A simple calculator that performs basic arithmetic operations.

---

## Concepts Used
- Input and Output
- Conditional Statements (if/else)
- Loops (for, while)
- Functions
- Basic Error Handling

---

## How to Run

Run each file using Python:

```bash
python level1_reverse_string.py
python level2_vowel_count.py
